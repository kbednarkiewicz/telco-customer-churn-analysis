# Telco Customer Churn Analysis 

This project focuses on exploratory data analysis (EDA) of customer churn in a telecommunications company.

The goal was to identify patterns and characteristics of customers who decided to leave the company.
The project was completed as part of a university course. The report was written in Polish.

## Authors

- Katarzyna Bednarkiewicz
- Ewelina Dąbrowa

## Dataset

The project uses the **Telco Customer Churn Dataset**, which contains information about customers of a telecommunications company.
Data source:  
[Kaggle - Telco Customer Churn Dataset](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)

## Analysis

The analysis included:

- data preparation,
- descriptive statistics,
- visualization of variable distributions,
- analysis of relationships between features,
- comparison of churned and active customers,
- interpretation of results and formulation of business recommendations.
